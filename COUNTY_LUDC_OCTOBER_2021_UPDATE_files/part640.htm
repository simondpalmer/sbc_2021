35.56.090 - Filing an Application for a Demolition and Reclamation Permit

A. When application required. A permittee of a permitted land use that has not filed an application to defer abandonment in compliance with Section 35.56.040 (Filing an Application to Defer Abandonment), or who has filed and that application has been disapproved, shall file an application for a Demolition and Reclamation Permit.
