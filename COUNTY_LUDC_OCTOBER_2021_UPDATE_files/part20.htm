January 2020 Republished Development Code

The Development Code was republished in its entirety in January 2020 to reflect revisions to the Development Code resulting from the adoption of Ordinance No. 5093 (Case No. 19ORD-00000-00003, adopted 12/10/2019) by the Board of Supervisors. Key amendments include the deletion of regulations that would apply solely within the Coastal Zone, amendments to commercial telecommunications facilities, and 2019 general package amendments. See Appendix A for information on the affected Development Code sections.
